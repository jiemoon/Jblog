# Jblog

## ENV Require
- php 5.6+

## Install

### 1. Clone Project
```shell
git clone https://github.com/jiemoon/Jblog.git
```

### 2. Install Composer Require
```shell
cd Jblog
composer install
```

### 3. Install Node Module
```shell
yarn // or npm install
```
